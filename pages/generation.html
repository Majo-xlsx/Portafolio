Bootcamp Desarrollo Full-Stack Java

Junio 2025 - Septiembre 2025

Generation Colombia (Virtual)

Formación intensiva Full-Stack centrada en Java, principios de backend, bases de datos relacionales y buenas prácticas de ingeniería.

Temas y proyectos

Empecé con SCRUM, aprendiendo a organizar proyectos bajo metodologías ágiles. Luego trabajé en Python para afianzar la lógica de programación y en JavaScript, primero con ejercicios prácticos y después aplicándolo en páginas con HTML, CSS y Bootstrap. Complementé esta parte con el uso de Figma para prototipado de interfaces y diseño visual.

También reforcé el uso responsable de la inteligencia artificial, enfocándome en la protección de datos personales y en la creación de prompts efectivos.

Más adelante profundicé en Java, revisando constructores, estructuras de control y programación orientada a objetos. Con Spring Boot desarrollé servicios REST, definiendo endpoints y gestionando peticiones/respuestas.

Finalmente trabajé con MySQL, diseñando bases de datos relacionales (tablas, relaciones, claves primarias y foráneas) y conectándolas con el backend para aplicaciones completas.

Competencias adquiridas

  • Metodologías ágiles: SCRUM
  • Lógica de programación: Python y JavaScript
  • Frontend: HTML, CSS, Bootstrap, DOM en JavaScript
  • Diseño de interfaces: Figma
  • Backend: Java, POO, Spring Boot, APIs REST
  • Bases de datos: MySQL, diseño relacional, claves y consultas CRUD
  • Buenas prácticas: uso responsable de IA, Git/GitHub, debugging
