Bootcamp Desarrollo Full-Stack Java

Junio 2025 - Septiembre 2025

Generation Colombia (Virtual)

Formación intensiva Full-Stack centrada en Java, principios de backend, bases de datos relacionales y buenas prácticas de ingeniería.

Temas y proyectos

  • Frontend (HTML, CSS, JavaScript, Bootstrap)
    Maquetación semántica con HTML, estilos con CSS y Bootstrap para diseño responsivo. Interacción y manipulación del DOM con JavaScript (fetch, manejo de eventos, validaciones básicas).
  • Backend con Java (Spring Boot)
    Creación de servicios REST, manejo de peticiones/respuestas, control de dependencias, configuración de endpoints y pruebas básicas de integración. Se reforzaron principios de programación orientada a objetos y diseño de APIs.
  • Bases de datos (MySQL)
    Diseño y consulta de bases relacionales, normalización básica, joins y operaciones CRUD. Integración de bases de datos con el backend y prácticas de migración de datos.
  • Desarrollo completo y despliegue
    Trabajo con control de versiones (Git/GitHub), integración de front y back, construcción del proyecto final (API + UI) y prácticas de despliegue/local run para demostraciones.
  • Refuerzo de lógica y algoritmos
    Resolución de problemas, estructuras de control, razonamiento algorítmico y debugging; ejercicios prácticos que mejoraron la fluidez en pensamiento computacional.
